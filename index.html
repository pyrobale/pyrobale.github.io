مستندات پایروبله

کتابخانه پایتون برای تعامل با API ربات بله

نصب و راه‌اندازی

pip install pyrobale

شروع سریع

from pyrobale import Client

bot = Client("YOUR_BOT_TOKEN")

@bot.on_message
def handle_message(message):
    if message.text == "/start":
        message.reply_message("سلام! به ربات من خوش آمدید!")

bot.run()

کلاس‌های اصلی

Client

کلاس اصلی برای تعامل با API بله

متدها:

  • __init__(token: str, session: str = 'https://tapi.bale.ai', database_name='database.db')
  • run() - شروع دریافت آپدیت‌ها
  • safe_close() - توقف ایمن ربات
  • send_message(chat_id, text, parse_mode=None, reply_markup=None)
  • send_photo(chat_id, photo, caption=None, parse_mode=None, reply_markup=None)
  • send_document(chat_id, document, caption=None, parse_mode=None, reply_markup=None)
  • send_video(chat_id, video, caption=None, parse_mode=None, reply_markup=None)
  • send_audio(chat_id, audio, caption=None, parse_mode=None, reply_markup=None)
  • send_voice(chat_id, voice, caption=None, parse_mode=None, reply_markup=None)
  • send_location(chat_id, latitude, longitude, reply_markup=None)
  • send_contact(chat_id, phone_number, first_name, last_name=None, reply_markup=None)
  • send_invoice(chat_id, title, description, payload, provider_token, prices, photo_url=None, reply_markup=None)

Message

نمایانگر یک پیام در بله

ویژگی‌ها:

  • message_id - شناسه منحصر به فرد پیام
  • from_user - فرستنده پیام
  • date - تاریخ ارسال پیام
  • chat - چت مربوط به پیام
  • text - متن پیام
  • caption - عنوان برای پیام‌های رسانه‌ای
  • document - اطلاعات سند
  • photo - سایزهای موجود عکس
  • video - اطلاعات ویدیو
  • audio - اطلاعات صوت
  • voice - اطلاعات پیام صوتی
  • contact - اطلاعات مخاطب
  • location - اطلاعات موقعیت مکانی

Chat

نمایانگر یک مکالمه چت

ویژگی‌ها:

  • id - شناسه منحصر به فرد چت
  • type - نوع چت (خصوصی، گروه، ابرگروه، کانال)
  • title - عنوان چت
  • username - نام کاربری چت
  • description - توضیحات چت

User

نمایانگر یک کاربر بله

ویژگی‌ها:

  • id - شناسه منحصر به فرد کاربر
  • is_bot - آیا این کاربر ربات است
  • first_name - نام کاربر
  • last_name - نام خانوادگی کاربر
  • username - نام کاربری

صفحه کلیدها

MenuKeyboardMarkup

ایجاد صفحه کلید سفارشی با دکمه‌های منو

keyboard = MenuKeyboardMarkup()
keyboard.add(MenuKeyboardButton("دکمه ۱"))
keyboard.add(MenuKeyboardButton("دکمه ۲"), row=1)
message.reply_message("یک گزینه انتخاب کنید:", reply_markup=keyboard)

InlineKeyboardMarkup

ایجاد صفحه کلید درون‌خطی با دکمه‌های بازخورد

keyboard = InlineKeyboardMarkup()
keyboard.add(InlineKeyboardButton("کلیک کنید!", callback_data="button1"))
keyboard.add(InlineKeyboardButton("مشاهده وبسایت", url="https://example.com"), row=1)
message.reply_message("یک گزینه انتخاب کنید:", reply_markup=keyboard)

مدیریت رویدادها

  • @bot.on_message - مدیریت پیام‌های جدید
  • @bot.on_callback_query - مدیریت درخواست‌های بازخورد از صفحه کلید درون‌خطی
  • @bot.on_member_chat_join - مدیریت اعضای جدید چت
  • @bot.on_member_chat_leave - مدیریت خروج اعضا از چت
  • @bot.on_message_edit - مدیریت پیام‌های ویرایش شده
  • @bot.on_ready - مدیریت راه‌اندازی ربات
  • @bot.on_close - مدیریت خاموش شدن ربات
  • @bot.on_tick(seconds) - مدیریت رویدادهای دوره‌ای

پایگاه داده

پایگاه داده SQLite داخلی برای ذخیره اطلاعات

# دسترسی به پایگاه داده
db = bot.database

# ذخیره داده
db.write_key("user_123", {"points": 100})

# بازیابی داده
data = db.read_key("user_123")

# حذف داده
db.delete_key("user_123")
